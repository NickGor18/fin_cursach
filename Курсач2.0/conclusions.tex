\chapter*{Заключение}
\addcontentsline{toc}{chapter}{ЗАКЛЮЧЕНИЕ}
В данной курсовой работе был проведен обзор управления по прогнозирующей модели (MPC) на примере неоклассической модели экономического роста.

Было получено магистральное состояние, гарантирующее максимальную скорость экономического роста, по результатам использования запрограмированного алгоритма магистральное состояние было достигнуто для достаточно большого горизонта $ T $.
